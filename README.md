# 📊 Meta Ad Performance Dashboard

> **Power BI Dashboard** — Meta (Facebook & Instagram) ke advertising campaigns ka complete performance analysis ek jagah par.

---

## 🗂️ Dashboard Pages

### 📄 Page 1 — Overall Performance (All Metrics)

Is page mein **saari metrics ka overview** diya gaya hai jisme impressions se lekar purchases tak sab kuch cover hota hai. Yahan **Dynamic Slicer = All** set hai, matlab sabhi ad types aur campaigns ka combined data show ho raha hai.

#### 🔢 Top KPI Cards:
| Metric | Value |
|---|---|
| Impressions | 216.0K |
| Shares | 1.3K |
| Comments | 2.6K |
| Clicks | 25.4K |
| Engagements | 242.6K |
| Conversion Rate | 5.21% |
| Avg. Campaign Budget | $50.72K |
| Total Budget | $2.5M |
| CTR | 1175.57% |
| Engagement Rate | 112.35% |

#### 📊 Visuals:
- **By Gender (Donut Chart):** Engagements, Impressions, Purchases, Comments ka gender-wise breakdown
- **By Age (Bar Chart):** User age (13–50+) ke hisaab se Comments, Clicks, Impressions, Engagements ka distribution — 25–35 age group sabse zyada active
- **By Weekly (Stacked Bar):** Week 18–32 tak Carousel, Image, Stories, Video ad types ka weekly performance
- **By Country (Map):** Duniya ke alag-alag countries mein ad reach ka geographic view
- **By Calendar:** Month-wise weekly calendar jisme active days highlight hain
- **By Event Hour (Line Chart):** Din ke alag-alag ghanton mein Comments, Clicks, Impressions, Engagements, Purchases aur Shares ka pattern
- **Ad Type Table:** Image, Carousel, Stories, Video ka detailed breakdown — Clicks, Shares, Impressions, Engagements, CTR aur Purchase Rate ke saath

### 📄 Page 2 — Comments Deep Dive

Is page mein **sirf Comments metric** par focus kiya gaya hai. **Dynamic Slicer = Comments** set hai, isliye saare charts comments ke angle se filtered hain.

#### 🔢 Top KPI Cards (Comments Filter ke saath):
| Metric | Value |
|---|---|
| Impressions | 123.8K |
| Shares | 682.0 |
| Comments | 1.5K |
| Clicks | 14.7K |
| Engagements | 139.2K |
| Conversion Rate | 4.82% |
| Avg. Campaign Budget | $50.72K |
| Total Budget | $2.5M |
| CTR | 1186.21% |
| Engagement Rate | 112.41% |

#### 📊 Visuals:
- **Comment by Gender (Donut Chart):** Male (28%), Female (37%), All (36%) — teeno gender segments ka comments share
- **Comment by Age (Bar Chart):** Age-wise comment distribution — 20–35 age group mein peak comments
- **Comment by Weekly (Stacked Bar):** Week 18–32 tak har week mein Carousel, Image, Stories, Video ke comments
- **Comment by Country (Map):** Countries ke hisaab se comments ka geographic spread
- **Comment by Calendar:** Month-wise weekly calendar with active days
- **Comments by Event Hour (Line Chart):** Raat 8 baje ke baad comments mein noticeable spike — best posting time identify karne ke liye useful
- **Ad Type Table (Comments filtered):** Carousel, Stories, Video, Image — CTR aur Purchase Rate ke saath detailed view

4. **Explore** Dynamic Slicer se alag-alag metrics filter karein

## 📌 Key Insights

- 🎯 **Stories** ad type ka **CTR sabse zyada** (1175.06%) hai
- 👥 **Female audience** comments mein slightly aage (37%) hai
- ⏰ **Raat 8–10 baje** comments aur engagement peak par hote hain
- 📅 **Week 22–24** mein overall performance best rahi
- 💰 **Image ads** ka Purchase Rate sabse zyada (57.35%) hai
